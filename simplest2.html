筆記

1. 建立網誌須注意檔案名稱是否衝突


2. 測試多筆資料時，新建一個測試倉儲


3. solvespace 繪圖 :

* 完成伸長填料或除料後按shift + 3 ( 新建一個草圖 )

*指定基準面後按 2

*f3 : 等角視圖

*f2 : 正視圖

*繪製相切圓及使用其他功能時，需注意所繪製的線是否在正確的基準面上，否則會出現錯誤

4. solvespace 也能旋轉填料( shift + l )，點選旋轉軸後按下指令即可


5. ctrl + shift + t 追蹤點的位置，並在結束( ctrl + shift + s )後產生一個csv檔 ( 可用 SciTE 或 Excel 查看位置變化，便於結構運動時的運算 )


6. 測量線段 : 如果是以點對點，測量出來的可能是有經過一角度的偏轉得到的長度，如果想要測量兩平行線段，須以一線段的點及另一線段整段者作測量


7. 在旋轉或伸長材料的時候，如果有干涉就會發生錯誤，目前還不知道如何將干涉的線段切除


8. 繪製圓弧時，功能tangant arc 很常出現問題，目前只會將線段刪除重畫解決
